# Security Policy

## Supported Versions

We currently only support the latest major version of MetaBun with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Security is a high priority for MetaBun. Since this framework interacts directly with game servers via Metamod, vulnerabilities could potentially compromise server integrity or result in crashes (Denial of Service).

If you discover a security vulnerability within MetaBun, **please DO NOT open a public issue.**

Instead, please privately report it to the maintainers by emailing: **[Your-Email-Here/Maintainer Contact]** or by using GitHub's "Private vulnerability reporting" feature if enabled on the repository.

### What to include in your report:
- A detailed description of the vulnerability.
- Steps to reproduce the issue (code snippets, scripts, etc.).
- The version of MetaBun and the game server details.
- Potential impact of the vulnerability.

We will acknowledge receipt of your vulnerability report as soon as possible and strive to send you regular updates about our progress.
